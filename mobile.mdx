## Getting the app

The Swish mobile app is available on the Solana dApp Store. Search for **Swish** or download it directly from the store listing under Privacy and Security.

## Signing in

The mobile app supports the same sign-in methods as the web app:

- **X (Twitter)** — sign in with your X account
- **Solana Wallet** — connect to your Seeker wallet

<Note>
  If you use the same X account or wallet on both web and mobile, you'll access the same Swish account and balance.
</Note>

## Features

<Note>
  Multi-protocol routing (Auto across [Privacy Cash, MagicBlock, and Umbra](/privacy-protocols)) is currently available on the web app only. The mobile app routes all sends through Privacy Cash — the aggregator is coming to mobile next.
</Note>

The mobile app supports all the same features as the web app:

- **Send** — private USDC transfers to wallet addresses or X handles
- **Request** — generate and share payment request links
- **Send via Claim** — create claimable links with passphrase
- **Deposit** — view your wallet QR code and address
- **Withdraw** — gas-sponsored USDC transfers

## Deep linking

The mobile app supports deep links for request and claim pages:

- **Request links** (`swish.cash/r/...`) — open directly in the app if installed
- **Claim links** (`swish.cash/c/...`) — open directly in the app if installed

If the app isn't installed, these links open in the browser at swish.cash instead.

<Tip>
  Swish is built for Seeker — connect instantly with the built-in wallet, no extra apps needed.
</Tip>

## Wallet export

Wallet export is only available on the web app. If you tap Export in the mobile app, you'll be redirected to [swish.cash/p](https://swish.cash/p) to complete the process in your browser.
